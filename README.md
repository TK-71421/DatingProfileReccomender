# Dating Profile Recommender

Machine learning exploration of behavioral and profile data to predict compatibility between dating app users.

## Methods
- Feature engineering from profile and behavior data
- PCA visualization
- Similarity scoring models
- ROC evaluation

## Experiments
Experiment A and B test different similarity scoring approaches.

## Visualizations
- Feature importance
- Pairwise feature distributions
- PCA projections
- ROC curves

## Datasets Used:
- https://www.kaggle.com/datasets/andrewmvd/okcupid-profiles
- https://www.kaggle.com/datasets/keyushnisar/dating-app-behavior-dataset?resource=download
